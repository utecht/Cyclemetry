## What's New

### Export Format Dialog

Rendering now opens an export dialog where you choose your codec, with per-codec render-time estimates so you can trade quality against speed before you commit.

### New Metrics

- **Time of day** for value overlays, with automatic timezone detection and DST-correct offsets computed from the activity's GPS timestamps.
- **Lean angle** support for value-driven elements.
- **Course plot** overlay, alongside gauge refinements.
- **Letter spacing** control for text elements.

### Revamped Properties Panel

The right sidebar properties have been reorganized with a clearer hierarchy and a basic/advanced split, so common controls stay up front and advanced options are tucked away until you need them.

## Editor Improvements

- Metric dropdowns now only show metrics that actually have data in the loaded activity
- Collapsible Overlay Timeline and Scene sections in the left sidebar
- Added a mini map to the activity picker
- Higher-resolution zoom, plus zoom and reset-zoom polish
- Option to hide the video underlay
- Smoother opacity slider behavior and better dialog positioning

## Bug Fixes

- Allow uppercase file extensions in native file pickers
- Fixed Linux video preview via a local HTTP range-request server
- Fixed a render that could block during testing

## Upgrading

Download the latest `.dmg` (macOS), `.exe` installer (Windows), or `.deb`/`.AppImage` (Linux) from [GitHub Releases](https://github.com/walkersutton/cyclemetry/releases).
