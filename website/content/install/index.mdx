## Requirements

- A GPX file from your ride (exported from Garmin, Wahoo, Strava, etc.)
- A video file recorded during the same activity
- The timestamps in your GPX and video need to roughly align — Cyclemetry lets you fine-tune the offset

## What You Get

After rendering, Cyclemetry produces a video file with the overlay burned in — ready to upload to YouTube or edit further in your video editor of choice. Output is ProRes on macOS for lossless editing, or H.264 for direct sharing.

## Updates

Cyclemetry can check for app updates from the menu bar. Open the app and choose **Cyclemetry → Check for Updates**.

New releases should also be pushed to the app automatically. When an update is available, Cyclemetry is expected to show an update prompt on app open so you can install it without downloading a new installer manually. This automatic prompt still needs verification, so use **Check for Updates** if you want to confirm you are on the latest version.
