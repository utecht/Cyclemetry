# Install Cyclemetry

Cyclemetry is a native desktop application. Download the release for your platform, then follow the platform-specific notes below.

<InstallDownloadGrid />

<p className="install-download-note">Cyclemetry is available for macOS, Windows, and Linux from GitHub Releases.</p>

## Requirements

- A GPX file from your ride (exported from Garmin, Wahoo, Strava, etc.)
- A video file recorded during the same activity
- The timestamps in your GPX and video need to roughly align — Cyclemetry lets you fine-tune the offset

## What You Get

After rendering, Cyclemetry produces a video file with the overlay burned in — ready to upload to YouTube or edit further in your video editor of choice. Output is ProRes for high-quality lossless editing.

## Updates

Cyclemetry checks for updates automatically on launch and shows a banner if a new version is available. You can also check manually from **Cyclemetry → Check for Updates** in the menu bar.
