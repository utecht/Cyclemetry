# Brand Assets

Source logo and app icon artwork for Cyclemetry.

Treat this directory as the source of truth for brand images. Runtime delivery
copies still live where each surface expects them:

- `app/public/logo192.png` serves the in-app header/logo asset.
- `website/public/logo.png` serves the website favicon/nav logo.
- `src-tauri/icons/` contains generated platform icon outputs for the Tauri
  bundle.
